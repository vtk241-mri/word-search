# Privacy Policy

## 1. Scope

This privacy policy applies to the Word Search project started in 2025 and maintained by Rostyslav as an educational web application.

## 2. Data Processed

The project stores data locally in the user's browser:

- player name
- game settings
- game history and results
- cookie consent preferences

The project does not require account registration and does not send this data to a remote backend.

## 3. Legal Basis (GDPR)

- Necessary storage: legitimate interest in keeping the application functional
- Optional categories: user consent

Analytics and marketing categories are disabled by default and require explicit action in the cookie settings UI.

## 4. Cookie / Local Storage Categories

- Necessary: game state, player data, settings, history, consent status
- Analytics: reserved category for aggregated usage analytics, currently not connected to a third-party analytics service
- Marketing: reserved category for future advertising integrations, currently not used

## 5. Retention

All project data is retained only in the local browser storage until the user clears it or resets the stored values inside the application.

## 6. User Rights

Under GDPR principles, users may:

- withdraw optional consent
- request deletion by clearing browser storage
- inspect the stored information directly in the browser

## 7. Third Parties

The application currently uses third-party open-source libraries only for development and runtime functionality. Their licenses are documented in `THIRD_PARTY_LICENSES.md`.
