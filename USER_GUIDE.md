# User Guide

## Purpose

Word Search is an educational browser game for finding hidden words on a letter grid.

## How to Use

1. Open the start page.
2. Enter a player name or start without one.
3. Configure difficulty, board size, word count and timer in settings.
4. Select letters in order to form a word.
5. Press `Перевірити` to validate the selection.
6. Review results and history after the round ends.

## Cookie and Privacy Notes

- The app asks for cookie / local storage preferences.
- Necessary storage is required for the game to function.
- Optional categories can be changed later through the cookie settings button.

## Limitations

- The project is intended for educational and demonstration purposes.
- Game history is stored locally, not in the cloud.
- No guarantee is provided for uninterrupted availability or data persistence after browser cleanup.

## Acceptable Use

- Do not use the project for unlawful activity.
- Do not treat the project as a secure archival system.
- Respect the open-source license terms for redistribution and modification.
